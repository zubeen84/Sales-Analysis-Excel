## Overview

A comprehensive sales performance analysis built entirely in Microsoft 
Excel, featuring an interactive dashboard designed for executive 
reporting and quick business insights.

The project demonstrates how Excel's analytical toolset: Power Query 
for data transformation, pivot tables for aggregation, and slicers for 
interactive filtering can deliver production-ready business 
intelligence without specialist BI software.

## Key Findings

### Revenue drivers
- Top products: Quix Gift, Magnum Set, Delores Gift
- Top category: Colors (dominant revenue share)
- Top occasion: Holi, followed by Valentine's Day

### Geographic insights
- Highest order volume: Imphal and Handwar
- Regional concentration suggests targeted campaign opportunity

### Seasonal patterns
- Revenue peaks: December and February
- Revenue dips: March, May and June
- Hourly demand: Stable across all hours of day

### Strategic recommendations
- Prioritise inventory for Colors category and Holi-related products ahead of Q4
- Target Imphal and Handwar with regional marketing campaigns
- Investigate spring dip (March–June) to design seasonal promotions
  and reduce revenue gap

## Technical Approach

**Data cleaning and transformation (Power Query)**
- Raw data ingested and transformed using Power Query
- Column standardisation, data type correction, and null handling
- Reproducible refresh pipeline, data updates propagate automatically to all pivot tables and charts

**Analysis (Pivot Tables)**
- Multi-dimensional aggregation across product, region, time and occasion
- Calculated fields for revenue per order and delivery time metrics
- Structured for drill-down from summary to transaction level

**Dashboard (Excel)**
- Slicers connected to all pivot tables for synchronised filtering
- Chart formatting aligned to executive reporting standards
- Single-sheet dashboard layout for presentation and PDF export
